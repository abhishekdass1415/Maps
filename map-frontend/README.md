# Map Frontend

React + TypeScript frontend for the Map Platform.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- Interactive map with Leaflet
- Smart search with autocomplete
- Marker clustering for performance
- Keyboard navigation (Enter/Escape)
- Responsive design

## Tech Stack

- React 19.2.0
- TypeScript 5.9.3
- Leaflet 1.9.4
- Vite 7.2.4
